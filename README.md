# Portfolio — Página Estática

Sitio estático para presentar tu portfolio como Ingeniero de Software.

Cómo probar localmente:

```bash
# Desde la carpeta portfolio
cd "~/OneDrive - Institución Educativa SEK/INFORMATICA/portfolio"
# Servir con Python (macOS tiene Python 3):
python3 -m http.server 8000
# Luego abre en el navegador: http://localhost:8000
```

Ediciones sugeridas:
- Reemplaza `[Tu Nombre]` en `index.html`.
- Cambia el correo en `js/app.js` para que el formulario muestre tu email.
- Añade capturas y enlaces reales en la sección de proyectos.
